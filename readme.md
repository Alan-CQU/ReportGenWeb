## 录屏软件调研报告生成系统

基于Django搭建的Web平台，

需要的用户输入：

- 产品名字
- 产品URL
- 想在产品之间做对比的特性

系统在后台进行爬虫，语句提取。综合整理成一篇调研报告，给出网页上的展示和word版本的下载（word版本下载暂时未实现）。

目前版本v 0.0.3



**仍待优化的问题**

- 产品特性对比中，筛选算法比较简单，准确率无法达到100%
- 文本生成中，筛选算法比较简单， 内容相关性不如预期的高
